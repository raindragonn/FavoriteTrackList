`Coroutine` - 비동기 처리를 위해 사용하였습니다.

`Koin` - 의존성 주입을 위해 사용하였습니다.

`Glide` - 편한 이미지 로딩을 위해 사용하였습니다.

`Paging3` - 페이지네이션을 위해 사용하였습니다.
